# Screenshots

Add application screenshots here for documentation.

## Required Screenshots

1. **home.png** - Home page with hero section
2. **products.png** - Product listing page with grid layout

## How to Add Screenshots

1. Take screenshots of the running application
2. Save them in this directory with descriptive names
3. Update the main README.md to reference the images
4. Commit and push the images

## Image Guidelines

- Format: PNG or JPG
- Resolution: 1920x1080 for desktop, 375x812 for mobile
- File size: Keep under 500KB per image
- Use descriptive filenames
